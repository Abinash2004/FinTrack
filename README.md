# Personal Finance Tracker

A **mini web application** to record, categorize, and analyze personal income and expenses.  
Primary intention is to practice **TypeScript** and **PostgreSQL** integration within a modern web stack.

---

## Tech Stack
- **Frontend:** React + TypeScript  
- **Backend:** Node.js (Express) + TypeScript  
- **Database:** PostgreSQL (via Docker Compose)  
- **Auth:** OAuth 2.0 + JWT  

---

## Completed Tasks
- Docker setup for PostgreSQL   
- Implemented migration system for managing SQL schema files  
- Secure JWT authentication (access + refresh tokens)
- User signup with email OTP (Redis + Nodemailer)
- Google OAuth integration
- Automatic account linking for existing local users

---
